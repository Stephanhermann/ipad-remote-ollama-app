# Changelog

## v0.1.0 - 2026-03-17

Erste veroeffentlichte Version von `ipad-remote-ollama-app`.

### Enthalten

- iPad-App fuer den Zugriff auf einen entfernten Ollama-Server
- Chat mit optionalem Streaming
- Verwaltung mehrerer Serverprofile
- Bearer-Token- und Basic-Auth-Unterstuetzung
- Lokale Speicherung von Chats
- Import und Export von Chats als JSON
- Spracheingabe und Push-to-talk
- Bild-Upload fuer multimodale Modelle
- Markdown-Darstellung inklusive Codebloecken mit Copy-Button
- Verbindungstest fuer Serverprofile
- Onboarding, Banner, Suche und dunkles Chat-Layout
